An nb.ds.NamedData containing an array of floats storing a resource's data location in a data file and other useful informations. Its name is the resource's path relative to the project's resource directory.

DataFileInfo

An nb.ds.NamedData containing an array of floats describing a data file. Its name is the data file's path relative to the project's executable.

Represents an entry, which is a file or a directory in an nb.fs.NFileSystem.
